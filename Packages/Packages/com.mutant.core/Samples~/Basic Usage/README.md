# Basic Usage

这个 sample 演示：

1. 如何创建 CoreSettings
2. 如何在场景中挂载 CoreBootstrap
3. 如何手动注册模块
4. 如何发送和接收 EventBus 事件
5. 如何使用 StickyEventBus

## 使用步骤

1. 在 Package Manager 中导入 `Basic Usage`
2. 在场景中创建 `[MutantCore]`
3. 挂载 `CoreBootstrap`
4. 创建一个 `CoreSettings` 资源并拖入
5. 创建一个空物体挂载 `CoreTestEntry`
6. 运行场景并观察 Console
