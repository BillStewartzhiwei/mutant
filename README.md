# Mutant

A Unity game project.

## Requirements

- [Unity](https://unity.com/download) (see `ProjectSettings/ProjectVersion.txt` for the required version)

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/BillStewartzhiwei/Mutant.git
   ```
2. Open the project in Unity Hub by selecting the cloned folder.
3. Open the main scene and press **Play** to run the game in the editor.

## Building

Use **File → Build Settings** in the Unity Editor to configure and build for your target platform.
